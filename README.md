# AI Calculator

A modern, responsive, and feature-rich calculator built with **HTML**, **CSS**, and **Vanilla JavaScript** using an **AI-assisted development workflow**. The project showcases front-end development fundamentals, interactive JavaScript functionality, responsive UI design, and version control with Git and GitHub.

## Live Demo

**Vercel:** https://calculator-ai-beryl.vercel.app

---

## Features

### Standard Calculator
- Addition, subtraction, multiplication, and division
- Decimal calculations
- Clear (C) and Delete (⌫) functionality
- Error handling for invalid operations
- Keyboard support

### Scientific Functions
- Square root (√)
- Square (x²)
- Power (xʸ)
- Percentage (%)
- Trigonometric functions (`sin`, `cos`, `tan`)
- Logarithmic functions (`log`, `ln`)
- Factorial (`n!`)
- Pi (π)

### User Experience
- Calculation history
- Dark / Light mode toggle
- Responsive design for desktop, tablet, and mobile
- Glassmorphism-inspired interface
- Smooth animations and transitions

## Getting Started

Clone the repository:

```bash
https://github.com/rayyanahmadz/calculator_ai
```

Navigate to the project directory:

```bash
cd calculator-ai
```

Run the application by opening `index.html` in your preferred web browser.
